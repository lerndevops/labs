
# Git Lab 02: Installing and Opening a Repository in VS Code

Before you start working with Git, you need to have Visual Studio Code (VS Code) installed on your computer.

## How to Install VS Code

### Windows
1. Go to [https://code.visualstudio.com/](https://code.visualstudio.com/)
2. Click **Download for Windows**.
3. Run the downloaded installer and follow the setup instructions.
4. Once installed, you can launch VS Code from the Start menu.

### Mac
1. Go to [https://code.visualstudio.com/](https://code.visualstudio.com/)
2. Click **Download for Mac**.
3. Open the downloaded `.zip` file and drag **Visual Studio Code** to your **Applications** folder.
4. Launch VS Code from the Applications folder or using Spotlight search.

---

Once VS Code is installed, you can open your project folder as described below.

## Step-by-step Guide

1. **Open VS Code**
   - Launch Visual Studio Code from your applications or start menu.

2. **Open the Repository Folder**
   - Click on **File > Open Folder...** (or **Open...** on macOS).
   - Browse to the folder where your repository is located (the folder you cloned or initialized).
   - Select the folder and click **Open**.

3. **(Optional) Open the Integrated Terminal**
   - Go to **View > Terminal** or press <kbd>Ctrl</kbd>+<kbd>`</kbd> (backtick) to open the built-in terminal.
   - You can now run Git commands directly inside VS Code.

4. **(Optional) Use the Source Control Panel**
   - Click the **Source Control** icon on the left sidebar (or press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>G</kbd>).
   - Here you can see changes, stage files, and make commits using the VS Code interface.

## Tips
- You can drag and drop a folder onto the VS Code window to open it quickly.
- VS Code will automatically detect if the folder is a Git repository and enable Git features.

---

Now you are ready to work with your repository in VS Code!
